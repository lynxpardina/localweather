DESCRIPTION:

  The app uses two api queries to gather local information, and display the local weather conditions based on the location of the ip used to access the page.

  Clicking on temperature changes between Celsius and Fahrenheit, saving in localstorage the choosen value for later reference.

  Background images and background color of temperature's rectangle changes depending of the value of it. For <10  blue tone, >=10 & <25 green tone, and >25 red tone
